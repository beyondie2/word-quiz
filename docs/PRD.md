# 영단어 외우기 앱
## 1. 프로젝트 개요
### 1.1 프로젝트명
영단어 외우기 앱
### 1.2 프로젝트 목적
사용자가 주어진 영단어를 보고, 그 의미를 맞추게 하고, 관리자가 학습 과정을 관리할 수 있는 간단한 풀스택 웹 앱
### 1.3 개발 범위
- 단어 맞추기 화면(회원 확인 기능, 책 선택 기능, 장 선택 기능, 단어 맞추기 기능)
- 수행 확인 화면(틀린 단어 확인)
- 데이터를 생성/조회/수정/삭제할 수 있는 기능
## 2. 기술 스택
- 프런트엔드: HTML, CSS, 리액트, 자바스크립트
- 백엔드: Node.js, Express
- 데이터베이스: PostgreSQL
## 3. 기본 사항
- 프런트엔드와 백엔드를 따로 개발
- 기본적인 웹 기술만 사용
- 결제 기능은 제외
- 영어를 보고 한국어로 맞추고, 한국어를 보고 영어를 맞춤

## 4. 프런트엔드 UI 명세

### 4.1 단어 맞추기 화면

#### 4.1.1 화면 구성

| 영역 | 구성 요소 | 설명 |
|------|----------|------|
| 헤더 | 로고, 탭 메뉴 | 앱 타이틀과 화면 전환 탭 |
| 설정 바 | 사용자 인증, 설정 버튼 및 옵션 | 사용자 확인 및 학습 조건 설정 |
| 문제 영역 | 출제 단어 | 맞춰야 할 단어 표시 |
| 입력 영역 | 텍스트 입력 필드 | 정답 입력 |
| 하단 버튼 | 액션 버튼 | 학습 진행 제어 |

#### 4.1.2 헤더 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| WORD TEST | 텍스트(로고) | 앱 타이틀, 좌측 배치 |
| 단어 맞추기 | 탭 버튼 | 현재 화면 (활성 상태 시 초록색 테두리) |
| 수행 확인 | 탭 버튼 | 수행 확인 화면으로 이동 |

#### 4.1.3 설정 바 영역

| 요소 | 타입 | 옵션 | 설명 |
|------|------|------|------|
| 이름 | 텍스트 입력 | - | 사용자 본인 인증을 위한 이름 입력 |
| 단어장선택 | 버튼 | - | 클릭 시 단어장 목록 표시 (모달/드롭다운) |
| 연습방법 | 라디오 버튼 | 한국어 / 영어 | 문제로 출제되는 언어 선택 |
| 한국어답 | 라디오 버튼 | 하나만 / 전부다 | 한국어 정답 입력 방식 선택 |
| 부분단원 | 버튼 | - | 클릭 시 단원 세부 선택 (모달/드롭다운) |

**이름 입력 필드 상세:**
- 플레이스홀더: "이름"
- 사용자 본인 인증 용도 (학습 기록 저장 시 사용자 식별에 사용)
- 학습 시작 전 필수 입력 권장

**연습방법 옵션 상세:**
- `한국어`: 한국어 단어를 보고 영어 뜻을 맞춤
- `영어`: 영어 단어를 보고 한국어 뜻을 맞춤

**한국어답 옵션 상세:**
- `하나만`: 여러 개의 한국어 뜻 중 하나만 입력해도 정답 처리
- `전부다`: 모든 한국어 뜻을 전부 입력해야 정답 처리

#### 4.1.4 문제 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| 출제 단어 | 텍스트 | 화면 중앙에 크게 표시, 맞춰야 할 단어 |

**표시 규칙:**
- 연습방법이 `영어`인 경우: 영어 단어 표시
- 연습방법이 `한국어`인 경우: 한국어 단어 표시

#### 4.1.5 입력 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| 정답 입력 필드 | 텍스트 입력 | 사용자가 정답을 입력하는 필드 |

**입력 필드 상세:**
- 플레이스홀더: 연습방법에 따라 동적 변경
  - `영어` 선택 시: "한국어_의미"
  - `한국어` 선택 시: "영어_의미"
- 입력 후 Enter 키 또는 제출 버튼으로 정답 확인

#### 4.1.6 하단 버튼 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| 틀린 것만 다시하기 | 버튼 | 이전에 틀린 문제들만 다시 출제 |

#### 4.1.7 사용자 흐름 (User Flow)

```
1. 화면 진입
   ↓
2. [이름] 입력 → 사용자 본인 인증
   ↓
3. [단어장선택] 버튼 클릭 → 단어장 목록에서 선택
   ↓
4. (선택) [부분단원] 버튼 클릭 → 특정 단원/범위 선택
   ↓
5. [연습방법] 선택 (한국어/영어)
   ↓
6. [한국어답] 옵션 선택 (하나만/전부다)
   ↓
7. 출제된 단어 확인
   ↓
8. 정답 입력 → Enter 또는 제출
   ↓
9. 정답/오답 피드백 표시
   ↓
10. 다음 문제로 이동 (반복)
   ↓
11. (선택) [틀린 것만 다시하기] 클릭 → 오답 문제 재출제
```

#### 4.1.8 상태 및 피드백

| 상태 | 시각적 피드백 |
|------|--------------|
| 정답 | 입력 필드 테두리 초록색, 정답 메시지 표시 |
| 오답 | 입력 필드 테두리 빨간색, 정답 표시 |
| 미입력 | 기본 상태 유지 |
| 이름 미입력 | 학습 시작 시 경고 메시지 표시 |

#### 4.1.9 디자인 가이드

**색상:**
- 기본 테마 색상: 초록색 (#4CAF50 또는 유사)
- 활성 탭/버튼 테두리: 초록색
- 오답 표시: 빨간색

**레이아웃:**
- 전체 컨테이너: 가운데 정렬, 최대 너비 제한
- 설정 바: 이름 → 단어장선택 → 연습방법 → 한국어답 → 부분단원 순서로 배치
- 각 영역 간 적절한 여백
- 모바일 반응형 지원

### 4.2 수행 확인 화면

#### 4.2.1 화면 구성

| 영역 | 구성 요소 | 설명 |
|------|----------|------|
| 헤더 | 로고, 탭 메뉴 | 앱 타이틀과 화면 전환 탭 |
| 필터 영역 | 사용자 선택, 날짜 선택 | 조회 조건 설정 |
| 콘텐츠 영역 | 수행 내용 목록 | 학습 수행 결과 표시 |

#### 4.2.2 헤더 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| WORD TEST | 텍스트(로고) | 앱 타이틀, 좌측 배치 |
| 단어 맞추기 | 탭 버튼 | 단어 맞추기 화면으로 이동 |
| 수행 확인 | 탭 버튼 | 현재 화면 (활성 상태 시 초록색 테두리) |

#### 4.2.3 필터 영역

| 요소 | 타입 | 설명 |
|------|------|------|
| 사용자 이름 | 드롭다운/버튼 | 조회할 사용자 선택 |
| 과제한 날짜 | 날짜 선택기 | 조회할 날짜 또는 날짜 범위 선택 |

**필터 동작:**
- 사용자 이름 선택 시: 해당 사용자의 수행 내용만 표시
- 과제한 날짜 선택 시: 해당 날짜에 수행된 내용만 표시
- 두 필터 조합 가능: 특정 사용자의 특정 날짜 수행 내용 조회

#### 4.2.4 콘텐츠 영역 (수행 내용)

| 요소 | 타입 | 설명 |
|------|------|------|
| 수행 내용 박스 | 컨테이너 | 학습 수행 결과를 표시하는 영역 |

**수행 내용에 포함될 정보:**
- 학습한 단어장 이름
- 학습한 단원/범위
- 학습 일시
- 총 문제 수
- 정답 수 / 오답 수
- 정답률
- 틀린 단어 목록

**틀린 단어 목록 표시 형식:**

| 항목 | 설명 |
|------|------|
| 영어 단어 | 틀린 문제의 영어 단어 |
| 한국어 뜻 | 해당 단어의 정답 (한국어 뜻) |
| 사용자 입력 | 사용자가 입력한 오답 |

#### 4.2.5 사용자 흐름 (User Flow)

```
1. 화면 진입
   ↓
2. (선택) [사용자 이름] 선택 → 특정 사용자 필터링
   ↓
3. (선택) [과제한 날짜] 선택 → 특정 날짜 필터링
   ↓
4. 수행 내용 목록 확인
   ↓
5. 특정 수행 기록 클릭 → 상세 내용 확인 (틀린 단어 목록 등)
```

#### 4.2.6 데이터 표시 상태

| 상태 | 표시 내용 |
|------|----------|
| 데이터 있음 | 수행 내용 목록 표시 |
| 데이터 없음 | "수행 기록이 없습니다" 메시지 표시 |
| 로딩 중 | 로딩 인디케이터 표시 |

#### 4.2.7 디자인 가이드

**색상:**
- 기본 테마 색상: 초록색 (#4CAF50 또는 유사)
- 활성 탭 테두리: 초록색
- 정답률 표시: 높음(초록), 중간(노랑), 낮음(빨강)

**레이아웃:**
- 필터 영역: 상단 좌측 정렬
- 콘텐츠 영역: 필터 아래 전체 너비 사용
- 수행 내용 박스: 둥근 모서리, 테두리 스타일
- 모바일 반응형 지원
